Design System

The System

The living reference for davidajudua.com. Every value below is a token defined in css/style.css and consumed by the site — one source of truth for colour, type, spacing, radii, elevation, frost, and motion. The Set (the fixed video) stays the aesthetic; this page just names the parts staged on top of it.
